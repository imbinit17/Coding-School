51 Java Programs Class IX ICSE

Here we are presenting you 51 miscellaneous programming questions in Java in particular following the ICSE class IX as of 2022 and special care has been taken account into keep the code and question in the scope of the syllabus . The basic concepts of OOP , along with general input/output using Scanner ,Math class , and at last the flow of control using loops are being used below .Below are the programs :-

  • Write a program to accept a number and check whether the number is palindrome or not .
  • Write a program to accept a number and check whether the number is a Perfect Number or not .
  • Write a program to accept a number and check whether the number is an Automorphic Number or not .
  • Write a program to calculate sum of the digits entered by the user .
  • Write a program to print the sum of negative numbers , sum of odd positive numbers and sum of even positive numbers from a list of numbers entered by user .The list terminates when the user enters a zero
    Note : Avoid array as out of scope of the ICSE class IX syllabus
  • Write a program to accept a number and check whether it is a special number or not
