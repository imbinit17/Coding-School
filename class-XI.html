51 Java Programs Class X ICSE

Here we are presenting you 51 miscellaneous programming questions in Java in particular following the ISC class XI as of 2022 and special care has been taken account into keep the code and question in the scope of the syllabus .All the general concepts learned in class IX and X are being used along with those of the XIth .

  • Write a class to accept a number and check whether the number is a Dissarium number or not .
  • Design a class PetersonNum to check whether a number is a Peterson number or not
  • Write a program to accept a number and check whether it is a tech number or not .
  • Design a class decimalToBinary to accept a non-zero number in decimal format and convert it to binary.
    Use of Strings and Arrays is prohibited.
  • Design a class PrimePalinGen to generate prime palindrome numbers.
    [ A number is said to be prime palindrome if the number is a prime as well as a palindrome number]
  • Design a class WordWise to separate words from a sentence and find the frequency of the vowels in each word.
    Define the class WordWise giving details of the constructor(), void readsent(), int freq vowel(String) and void arrange(). Define the main() function to create an object and call the functions accordingly to enable the task.
